# ESLint Config (wasdwave-eslint)

Modern ESLint 9+ configuration with Flat Config support for JavaScript, TypeScript, Vue 3, and React.

## ✨ Features

- 🔥 **ESLint 9+** with new Flat Config format.
- 📦 **JavaScript/TypeScript** support out of the box.
- 🖖 **Vue 3** support (SFC + TypeScript).
- ⚛️ **React** support (Hooks + JSX).
- 🛠️ **Best Practices**: Includes a built-in rule for event handlers (must start with `handle`).

## 👀 How to use?

In your `eslint.config.js`:

```javascript
import ESLintConfig from 'wasdwave-eslint'

export default [...ESLintConfig]
```
